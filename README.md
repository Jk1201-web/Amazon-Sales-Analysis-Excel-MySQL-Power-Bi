# Amazon Sales Data Analysis

## Project Overview
This project analyzes Amazon sales data using MySQL and Power BI
to extract business insights like revenue, order trends, and
customer behavior.

## Tools Used:
- MySQL
- Power BI
- CSV Dataset
- GitHub

## Business Questions Answered
- Total Revenue
- Total Orders
- Average Order Value
- Sales by Category
- Monthly Sales Trend
- Most Used Payment Methods
- Order Status Distribution

## Dashboards
### 1. Sales Overview Dashboard
- Revenue KPIs
- Category-wise Sales
- Monthly Trends

### 2. Order Insights Dashboard
- Order Status
- Payment Methods
- Customer Location
